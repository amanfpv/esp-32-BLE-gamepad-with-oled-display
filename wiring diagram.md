LEFT JOYSTICK
----X → GPIO 32
----Y → GPIO 33
----VCC → 3.3V
----GND → GND

RIGHT JOYSTICK
----X → GPIO 34
----Y → GPIO 35
----VCC → 3.3V
----GND → GND

JOYSTICK BUTTONS
----L3 → GPIO 4
----R3 → GPIO 16


ABXY BUTTONS
----A → GPIO 27
----B → GPIO 26
----X → GPIO 25
----Y → GPIO 13
----Other leg of each → GND

D-PAD BUTTONS
----UP → GPIO 18
----DOWN → GPIO 19
----LEFT → GPIO 23
----RIGHT → GPIO 5
----Other leg of each → GND

SHOULDER BUTTONS
----LB → GPIO 17
----RB → GPIO 14
----Other leg of each → GND

OLED (SH1106)
----SDA → GPIO 21
----SCL → GPIO 22
----VCC → 3.3V
----GND → GND
